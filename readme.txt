== Installation ==

1. Upload `wordpresstwitterplugin` to the `/wp-content/plugins/` directory or upload .zip from WP Admin Panel and activate it.
2. Open dev.twitter.com, create your own app and get API keys (consumer_key, auth_token etc.)
3. Go to twitter widget settings and fill the AUTH settings. 
4. Enjoy!
